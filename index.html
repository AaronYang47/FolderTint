Folder Color Changer

Advanced macOS Folder Customization Tool

Overview

Folder Color Changer is a sophisticated macOS application that allows users to customize folder icons with a wide range of colors. Built with SwiftUI, it offers an intuitive and modern interface for personalizing folder appearances while maintaining the native macOS look and feel.

Key Features

Multi-Language Support

  • 15 languages supported
  • Dynamic language switching
  • Real-time interface updates

Color Selection

  • 13 curated color categories
  • 100+ preset colors
  • Custom color support

Smart Naming System

  • Automatic color name appending
  • Duplicate handling
  • Original name preservation

Color Categories

Basic Colors

Red, Blue, Green, Yellow, Purple, Orange

Metallic Colors

Silver, Gold, Bronze, Copper, Platinum, Steel, Chrome, Titanium

Gem Colors

Ruby, Sapphire, Emerald, Amethyst, Topaz, Opal, Pearl, Garnet

Neon Colors

Pink, Blue, Green, Yellow, Orange, Purple, Cyan, Red

Technical Details

  • Built with SwiftUI and AppKit integration
  • Uses Core Image for color processing
  • Implements NSWorkspace for icon manipulation
  • Efficient memory management
  • Asynchronous operations for smooth performance

System Requirements

  • macOS operating system
  • Sufficient disk space for folder duplication
  • Support for modern color processing
